# **App Name**: Bingo Blitz

## Core Features:

- Board Generation: Generate a random 5x5 Bingo board with unique numbers.
- Number Calling: Randomly select and call out numbers.
- Marking Numbers: Allow users to mark numbers on their board when called.
- Bingo Detection: Automatically detect if a player has achieved Bingo (5 in a row).
- AI Pattern Detection: An AI tool to detect common Bingo patterns on generated boards.

## Style Guidelines:

- Primary color: Royal blue (#4169E1) for a classic Bingo feel.
- Background color: Light blue (#E6F0FF), a very desaturated version of the primary, for a calming background.
- Accent color: Purple (#800080), to highlight called numbers.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, machined look; good for both headers and body
- Simple, clear icons for game actions (e.g., restart, settings).
- Clean and organized layout with the Bingo board as the central focus.
- Subtle animations for number calls and Bingo announcements.
